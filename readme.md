# A Compiler

## Compilation Modes
The compiler now supports several modes:
- **release**: Standard compilation mode.
- **debug**: Includes debug information for debugging.
- **debug_opt**: Includes optimizations along with debug information.
- **debug_raw**: Generates raw output for further analysis.

## GCC Flags
| Flag          | Description                          |
|---------------|--------------------------------------|
| -O2           | Optimization level 2                 |
| -g            | Include debug information             |
| -Wall         | Enable all compiler's warning messages|
